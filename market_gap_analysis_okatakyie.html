Snack Market Gap Analysis¶

Business Objective¶

The goal of this project is to identify underserved opportunities in the snack market using nutritional data from OpenFoodFacts.

Rather than simply describing existing products, the project aims to answer:

If a company wanted to launch a new snack product today, what nutritional profile would offer the strongest opportunity?

The analysis combines:

  • Data engineering
  • Feature engineering
  • Market segmentation
  • Nutritional intelligence
  • Unsupervised machine learning

Dataset: OpenFoodFacts Product Database

Phase 1 — Data Ingestion¶

The OpenFoodFacts database contains millions of products.

To efficiently process the dataset:

  • Only required columns were loaded
  • Data was streamed in chunks
  • Snack-related products were filtered early

This reduced memory consumption while focusing the analysis on relevant products.

def is_snack(category_text):
    if pd.isna(category_text):
        return False

    text = str(category_text).lower()

    return any(keyword in text for keyword in SNACK_KEYWORDS)

Cleaning steps:

  • Removed products missing key nutritional information
  • Imputed missing fiber values using the median
  • Removed impossible nutritional values

The objective was to create a reliable analytical dataset before feature engineering.

Phase 3 — Category Engineering¶

OpenFoodFacts contains thousands of detailed category labels.

To improve interpretability, products were grouped into high-level snack categories:

  • Protein Bars
  • Nuts
  • Cookies
  • Chips
  • Candy & Chocolate
  • Crackers
  • Popcorn
  • Pretzels
  • Other Snacks

This creates business-friendly market segments.

def assign_category(text):

    if pd.isna(text):
        return "other_snacks"

    text = text.lower()

    if "protein bar" in text:
        return "protein_bars"

    elif "cookie" in text or "biscuit" in text:
        return "cookies"

    elif "chips" in text or "crisps" in text:
        return "chips"

    elif "cracker" in text:
        return "crackers"

    elif "popcorn" in text:
        return "popcorn"

    elif "pretzel" in text:
        return "pretzels"

    elif "nuts" in text:
        return "nuts"

    elif (
        "chocolate" in text
        or "candy" in text
        or "confectionery" in text
        or "caramel" in text
    ):
        return "candy_chocolate"

    return "other_snacks"

Candidate's Choice Feature: Nutrition Opportunity Score (NOS)¶

The project requirements asked for identifying market opportunities through visualization.

However, visual inspection alone does not quantify opportunity.

To address this limitation, a custom Nutrition Opportunity Score (NOS) was developed.

NOS combines:

  • Protein content
  • Fiber content
  • Sugar content
  • Category saturation

into a single metric.

This allows categories to be ranked according to both nutritional attractiveness and competitive density.

The score provides a quantitative framework for identifying potentially underserved product segments.

— Ingredient Intelligence¶

Analysis of products located in the High Protein + Low Sugar opportunity region revealed several recurring protein sources.

The most common ingredients were soy, peanuts, whey, almonds, eggs, and cashews. These ingredients appear repeatedly across nutritionally attractive products and therefore represent key formulation strategies used by existing manufacturers.

The results suggest that successful high-protein snacks rely on a combination of traditional protein fortification ingredients (soy, whey) and whole-food protein sources (nuts, oats, eggs). This insight can guide future product development by identifying ingredients that already demonstrate market acceptance.

Phase 8 — K-Means Market Segmentation¶

K-Means clustering was used to identify naturally occurring nutritional segments.

Features used:

  • Energy
  • Fat
  • Sugar
  • Fiber
  • Protein

Cluster selection was based on:

  • Elbow Method
  • Silhouette Analysis

Four clusters were selected because they provided meaningful business segmentation while maintaining acceptable separation.
